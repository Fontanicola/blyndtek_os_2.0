# Blyndtek OS

Proyecto base de Blyndtek OS sobre Next.js 14, TypeScript estricto, Tailwind CSS y Supabase.

## Setup

1. Cloná el repositorio.
2. Ejecutá `npm install`.
3. Copiá `.env.example` a `.env.local` y completá las variables necesarias.
4. Ejecutá `npm run dev`.
